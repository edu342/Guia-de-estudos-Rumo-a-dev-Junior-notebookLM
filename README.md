 ##  Miniguia de Estudo 

Este guia consolida os principais conhecimentos obtidos através do caderno temático, servindo como base de consulta rápida e revisão.

### 📌 1. Resumos Estruturados do Assunto
Com base na curadoria de fontes e interações com a IA, o caminho de evolução foca em três pilares interdependentes:
* **O javascript no Frontend:** Bom o javascript ele deixa as paginas interativas atraves do DOM que manipula o html e faz o codigo JS interagir com o HTML
* **Lógica de Servidor Backend:** O Node.js ele vai rodar diretamente no servidor ele interage diretamente com o banco de dados Lidando com toda logica por tras do Front-end.
* **Persistência de Dados:** a importancia é que dados salvos em variaveis são temporarios, e deletadas quando servidor e atualizado tornando-se inviavel em sistemas reais, por isso, usamos um backend com um banco de dados para gerenciar e centralizar essas informações de forma permanente.
* 
###  2. Glossário de Conceitos Chave
* **DOM (Document Object Model):** Estrutura que representa a página HTML como um objeto, permite que o Javascript interaja com o HTML assim podendo alterar estilos e adicionar respostas na nossa pagina.
* **Event Loop:** Mecanismo do Node.js que permite operações de entrada e saida , vai gerenciar com e quando as tarefas serão execultadas.
* **Middleware:** funções que tem acesso a objetos de requisições e respostas, serve para validar, e autenticar os usuarios
